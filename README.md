# my-starter-kit

Claude Code 설정 스타터 킷입니다.

## 구성

### MCP 서버 (`.mcp.json`)
- `context7` — 최신 라이브러리 문서 조회
- `playwright` — 브라우저 자동화
- `sequential-thinking` — 단계별 사고 지원

### 커스텀 에이전트 (`.claude/agents/`)
- `code-quality-reviewer` — 코드 품질/보안/성능 리뷰

### 커스텀 명령어 (`.claude/commands/`)
- `add-component` — React 함수형 컴포넌트 템플릿 생성
